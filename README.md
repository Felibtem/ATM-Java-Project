# ATM Java Project

This is a simple Java ATM project demonstrating object-oriented programming.

## Features
- Create Bank Accounts
- Deposit Money
- Withdraw Money
- Show Account Balance

## Classes
1. **BankAccount.java** – Represents a bank account
2. **Bank.java** – Manages multiple accounts
3. **Main.java** – Runs the ATM menu interface
